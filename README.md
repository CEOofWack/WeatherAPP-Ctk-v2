# WeatherAPP-Ctk-v2

A Python desktop weather application with a GUI that retrieves real-time weather data from the OpenWeatherMap REST API.

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/CEOofWack/WeatherAPP-Ctk-v2.git
cd WeatherAPP-Ctk-v2
pip install -r requirements.txt

```
Get a free OpenWeatherMap API key from openweathermap.org/api and create a python enviroment variable called OPENWEATHER_API_KEY or add to line 7 on 'your_api_key_here'. 
Download weather icon images as needed for the application.
